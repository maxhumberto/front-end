Máquina de Refrigerantes em Python

Desenvolvi uma simulação em Python de uma máquina de refrigerantes que entregava um refrigerante para cada moeda inserida. O projeto foi fundamentado em conceitos de Teoria da Computação e Compiladores, utilizando lógica de processamento sequencial para reconhecer e responder à inserção de moedas. A aplicação demonstrou como autômatos e transições de estados podem ser aplicados em sistemas computacionais do cotidiano, proporcionando uma abordagem prática para o estudo de linguagens formais e compilação. Esse projeto consolidou minha habilidade em aplicar teoria computacional em soluções reais.

Identificação de Tipos de Pele com IA e Realidade Virtual

Desenvolvi um projeto de Inteligência Artificial (IA) e Computação Gráfica/Realidade Virtual para identificar diferentes tipos de pele (oleosa, seca, etc.) e recomendar tratamentos personalizados. Utilizando técnicas de aprendizado de máquina, treinei a IA com uma base de dados de imagens de pele, aplicando métodos de classificação e reconhecimento para diagnosticar o tipo de pele de um usuário. A recomendação de tratamento foi feita com base nos resultados da identificação. Esse projeto combinou IA com realidade virtual para criar uma experiência interativa e personalizada para os usuários, demonstrando como a tecnologia pode ser aplicada à dermatologia e cuidados pessoais.

Aplicativo de Compras - Design de Interface (Figma, UX/UI)

Desenvolvi um aplicativo de compras com foco em uma experiência de usuário intuitiva e atraente, utilizando metodologias de design como Figma, UX e UI. O projeto envolveu o design completo da interface, com protótipos interativos e fluxos de navegação otimizados para facilitar a jornada do usuário, desde a escolha do produto até o pagamento. A aplicação dos princípios de UX e UI garantiu um visual limpo, funcional e responsivo, enquanto o uso do Figma possibilitou uma prototipagem eficiente, permitindo testar e validar a experiência antes da implementação. Esse projeto fortaleceu minha habilidade em criar interfaces que não apenas atendem às necessidades dos usuários, mas também oferecem uma navegação agradável e fluida.
